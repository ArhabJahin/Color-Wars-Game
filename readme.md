# 🎮 Color Wars

**Color Wars** is a modern online multiplayer strategy game inspired by the classic **Chain Reaction** board game. Players compete by strategically placing colored orbs on a grid, triggering explosive chain reactions to capture opponents' cells and dominate the entire board.

This repository contains the complete development history of the project, including multiple prototype versions and the latest implementation.

---

# ✨ Features

## 🎲 Gameplay

- Chain Reaction inspired mechanics
- 2–4 player support
- Human vs Human
- Human vs AI
- Online PvP multiplayer
- Multiple board sizes
- Multiple difficulty levels
- Smooth chain reaction animations
- Dynamic turn system
- Player elimination system
- End-game ranking screen
- Play Again system

---

## 🤖 Adaptive AI

Color Wars includes an advanced AI system featuring:

- Multiple AI difficulty levels
- Adaptive learning
- Monte Carlo Tree Search (MCTS)
- Policy-Value evaluation model
- Opening strategy memory
- Replay buffer
- Opponent profiling
- Federated learning synchronization
- Reinforcement learning controller
- Position evaluation heuristics

The AI continuously improves its decisions based on gameplay patterns while remaining lightweight enough to run efficiently.

# 🚀 Getting Started

## Clone the repository

```bash
git clone https://github.com/ArhabJahin/Color-Wars.git
```

## Install server dependencies

```bash
cd server
npm install
```

## Start the server

```bash
npm start
```

Then open the client in your browser.
